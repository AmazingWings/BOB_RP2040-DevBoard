## Welcome!
Hi! This is my first SMT board! It is a custom RP2040 development board named BOB. I chose BOB as the name because I like Bob; it's a fun name to say. But on the other hand, it is designed to act as a platform for me to learn SMT board design. The reason why I made it was not only to learn how to make a fully SMT board, but to serve as a platform for learning for my future flight computer designs I will do for my rockets. I wanted to transition away from using software like EasyEDA to more professional software like KiCad, which is much more widely used. Through this project, I not only learned SMT schematic design but also routing. In EasyEDA, I used to autoroute my components, but in KiCad and through this project, I learned how to route components and place components by myself, transitioning away from my dependencies with breakout boards. 

### Key features:
1. Dual-core processor with up to 133 MHz in speed.
2. 264 KB of on-chip SRAM.
3. Programmable through USB-C.
4. 30 GPIO pins that can run I2C, SPI, and UART. 
5. 2-layer PCB design: top layer for components and bottom layer for ground wiring and signal routing, along with the top layer for signal wiring and power wiring.
